# TabBar 图标说明

## 所需图标列表

请准备以下图标文件（建议使用 81x81 像素的 PNG 格式）：

| 文件名 | 描述 | 颜色 |
|--------|------|------|
| home.png | 首页图标（未选中） | #86868b |
| home-active.png | 首页图标（选中） | #0071e3 |
| card.png | 权益卡图标（未选中） | #86868b |
| card-active.png | 权益卡图标（选中） | #0071e3 |
| order.png | 拼班图标（未选中） | #86868b |
| order-active.png | 拼班图标（选中） | #0071e3 |
| user.png | 我的图标（未选中） | #86868b |
| user-active.png | 我的图标（选中） | #0071e3 |
| marker.png | 地图标记图标 | 蓝色 |

## 图标设计建议

- 风格：线性图标，符合 Apple 设计风格
- 线宽：建议 2-3px
- 尺寸：81x81 像素（@2x 为 162x162）
- 格式：PNG，透明背景

## 临时方案

在没有图标文件的情况下，TabBar 会使用文字显示。建议尽快添加图标以提升用户体验。
