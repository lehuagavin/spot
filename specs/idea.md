# Sports On-site Training

## Overview

The sports on-site training platform is a WeChat mini program platform that allows users to book qualified coaches for sports training courses.

## Features

- 管理员可以发布拼班课程，用户可以查看课程，并选择课程进行报名

- 首次进入会弹出隐私保护提示，参考，![隐私](./ui/secret.png)

- 同意隐私后进入地理位置授权界面，参考 ![地理](./ui/position.png)

### 首页

- 首页顶部是轮播图，下面是全部、进行中、已结束的班级列表,参考 ![首页](./ui/homepage.png)

- 首页中支持地图选择小区，点击进入地图界面，参考 ![地图](./ui/map.png)

- 点击地图界面的搜索框，跳转到小区列表界面，这些小区支持管理员配置 ，参考 ![地图](./ui/map1.png)

- 点击首页中的班级可以查看班级详情，参考 ![班级详情](./ui/course-detail1.png) 和 ![班级详情](./ui/course-detail2.png)

- 点击班级详情中加入拼班按钮，可以进入支付页面，参考 ![拼班信息](./ui/pay.png) 和 参考 ![拼班信息](./ui/pay1.png)

- 点击支付页面中【选择加入拼班的学员】按钮，可以进入选择学员管理页面，参考 ![选择学员](./ui/student-manage.png) 和 ![选择学员](./ui/student-manage1.png)

- 学员管理页面新增学员进入新增学员界面，参考 ![新增学员](./ui/add-student.png)

- 点击支付页面中确认支付，进入确认支付页面，参考  ![确认支付](./ui/pay2.png) 和 ![确认支付](./ui/pay3.png)
